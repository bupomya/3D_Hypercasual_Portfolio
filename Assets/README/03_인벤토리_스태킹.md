# 인벤토리 & 스태킹 시스템

아이템을 시각적으로 쌓아 보여주는 인벤토리 시스템입니다.
두 가지 방식의 스택커가 있으며, 각각 다른 용도로 사용됩니다.

```
StackerType (열거형) ← 아이템 종류 구분
ItemStacker         ← 수직으로 쌓기 (플레이어 머리 위)
GridStacker         ← 격자로 쌓기 (드롭 존 바닥)
```

---

## StackerType.cs (열거형)

아이템의 종류를 구분하는 열거형입니다.

```csharp
enum StackerType {
    Mineral,      // 일반 광물
    MineralRed,   // 빨간 광물
    Money          // 돈
}
```

`PlayerMiner.GetStacker(StackerType type)`에서 이 열거형으로 적절한 스택커를 찾습니다.

---

## ItemStacker.cs (수직 스태킹)

아이템을 **세로로 한 줄씩 쌓는** 인벤토리입니다.
주로 플레이어 캐릭터 머리 위에 광물이나 돈을 쌓는 데 사용됩니다.

### 주요 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `itemPrefab` | GameObject | 추가 시 생성할 아이템 프리팹 |
| `stackInterval` | float | 아이템 사이 수직 간격 (기본값 0.3) |
| `maxCount` | int | 최대 보유 가능 수량 (기본값 99) |
| `itemRotation` | Vector3 | 아이템의 회전값 |
| `removeDuration` | float | 제거 시 축소 애니메이션 시간 (기본값 0.3초) |

### 주요 메서드

| 메서드 | 설명 |
|---|---|
| `AddItem()` | `itemPrefab`을 인스턴스화하여 맨 위에 추가합니다. |
| `AddItem(GameObject item)` | 이미 존재하는 아이템을 스택에 추가합니다. |
| `RemoveItem()` | 맨 위 아이템을 **축소 애니메이션 후 파괴**합니다. (DOTween InBack) |
| `PopItem()` | 맨 위 아이템을 **애니메이션 없이 꺼내서 반환**합니다. (파괴하지 않음) |
| `RemoveAll()` | 전체 아이템을 즉시 제거합니다. |

### 주요 프로퍼티

| 프로퍼티 | 설명 |
|---|---|
| `int Count` | 현재 보유 아이템 수 |
| `bool IsFull` | 가득 찼는지 여부 (`Count >= maxCount`) |

### 아이템 배치 방식

```
[아이템 3] ← Y = stackInterval × 2   (맨 위)
[아이템 2] ← Y = stackInterval × 1
[아이템 1] ← Y = 0                    (맨 아래)
━━━━━━━━━━
 스택커 위치
```

추가된 아이템은 스택커 Transform의 자식이 되어 캐릭터를 따라다닙니다.

## GridStacker.cs (그리드 스태킹)

아이템을 **가로·세로 격자 형태**로 배치하는 인벤토리입니다.
주로 DropZone이나 PickupZone의 바닥에 아이템을 쌓는 데 사용됩니다.

### 주요 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `itemsPerRow` | int | 한 줄에 배치할 아이템 수 (기본값 3) |
| `rowSpacing` | float | 아이템 사이 수평 간격 (기본값 0.5) |
| `layerHeight` | float | 층 간 수직 간격 (기본값 0.3) |
| `maxCount` | int | 최대 보유 가능 수량 (기본값 0 = **무제한**) |

### 주요 메서드

| 메서드 | 설명 |
|---|---|
| `bool AddItem(GameObject item)` | 다음 빈 격자 위치에 아이템을 배치합니다. `IsFull`이면 `false` 반환. |
| `PopItem()` | 마지막 아이템을 꺼내서 반환합니다. (파괴하지 않음) |
| `GetNextPosition()` | 다음 아이템이 배치될 월드 위치를 반환합니다. |
| `GetStackPosition(int index)` | 특정 인덱스의 로컬 위치를 계산합니다. |

### 주요 프로퍼티

| 프로퍼티 | 설명 |
|---|---|
| `int Count` | 현재 보유 아이템 수 |
| `bool IsFull` | 가득 찼는지 여부 (`maxCount > 0 && Count >= maxCount`) |

### 배치 계산 방식

```
layer (층)            = index / itemsPerRow
positionInLayer (열)  = index % itemsPerRow
X 오프셋              = 중앙 정렬 기준으로 좌우 배치
Y 오프셋              = layer × layerHeight
```

예시 (itemsPerRow = 3):

```
[층 1]  [4] [5] [6]     ← Y = layerHeight × 1
[층 0]  [1] [2] [3]     ← Y = 0
```

## ItemStacker vs GridStacker 비교

| 항목 | ItemStacker | GridStacker |
|---|---|---|
| **배치 방식** | 수직 한 줄 | 가로×세로 격자 |
| **주 사용처** | 플레이어 머리 위 | 드롭/픽업 존 바닥 |
| **프리팹 생성** | `AddItem()`으로 자동 생성 가능 | 외부에서 생성한 아이템을 받음 |
| **제거 애니메이션** | DOTween 축소 효과 있음 | 없음 (즉시 반환) |
| **용량 제한** | `maxCount`로 제한 | `maxCount`로 제한 (0이면 무제한) |
