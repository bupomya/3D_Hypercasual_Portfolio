# 유틸리티 스크립트

게임의 핵심 로직을 보조하는 유틸리티 스크립트들입니다.

```
ItemTweenHelper  (정적) ← 아이템 포물선 애니메이션 + 존 시각화
TopDownCamera           ← 탑다운 카메라 추적
WarningUI    (싱글턴)   ← 임시 경고 메시지 표시
ClearUI      (싱글턴)   ← 레벨 클리어 메시지 표시
IdleHintUI   (싱글턴)   ← Idle 시 손가락 드래그 힌트 (→ 16_IdleHintUI_시스템.md)
MoneyItem    (마커)     ← 돈 아이템 식별용 컴포넌트
```

---

## ItemTweenHelper.cs (정적 유틸리티)

아이템 이동 애니메이션과 에디터 시각화 기능을 제공하는 **정적 클래스**입니다.
인스턴스를 만들지 않고 `ItemTweenHelper.메서드명()`으로 바로 호출합니다.

### 정적 메서드

#### `MoveArc(Transform item, Vector3 targetWorldPos, float duration, float height)`

아이템을 **포물선(아크) 경로**로 이동시킵니다.

- DOTween의 **CatmullRom 경로**를 사용합니다.
- 시작점과 끝점 사이의 중간 높이에 경유점을 생성하여 자연스러운 곡선을 만듭니다.
- Ease: OutQuad (처음 빠르고 점점 느려짐)
- 반환값: `Tween` 객체 (완료 콜백 등록 가능)

```csharp
// 사용 예시
ItemTweenHelper.MoveArc(item.transform, targetPos, 0.4f, 2f)
    .OnComplete(() => { /* 도착 후 처리 */ });
```

#### `DrawZoneGizmos(BoxCollider col, Transform transform, Color color)`

에디터 Scene 뷰에서 존 영역을 **반투명 색상 박스**로 시각화합니다.

- 각 TransferZone과 MiningZone의 `OnDrawGizmos()`에서 호출됩니다.
- 게임 실행 시에는 보이지 않고, 에디터에서만 표시됩니다.

### 사용처

거의 모든 아이템 이동에 사용됩니다:
- `DropZone` — 플레이어 → GridStacker 이동
- `PickupZone` — GridStacker → 플레이어 이동
- `ExchangeManager` — 광물 → NPC 이동
- `ConveyorLine` — 가공 완료 아이템 → 출력 스택커 이동

## TopDownCamera.cs

플레이어를 **탑다운/아이소메트릭 시점**으로 따라가는 카메라입니다.

### 주요 필드

| 필드 | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `target` | Transform | null (자동 탐색) | 추적 대상. null이면 "Player" 태그로 자동 탐색 |
| `height` | float | 15 | 카메라와 대상 사이의 거리 |
| `angle` | float | 45 | 카메라 수직 각도 (도) |
| `yRotation` | float | 0 | 카메라 수평 회전 (도) |
| `smoothSpeed` | float | 5 | 추적 부드러움 정도 |

### 동작 원리

- **구면 좌표계**를 사용하여 `angle`과 `height`로 카메라 위치를 계산합니다.
- `LateUpdate()`에서 `Lerp`로 부드럽게 따라갑니다.
  - `LateUpdate`를 쓰는 이유: 플레이어 이동(`Update`)이 끝난 뒤 카메라가 따라가야 떨림이 없습니다.

## WarningUI.cs (싱글턴)

화면에 임시 경고 메시지를 표시하고 자동으로 사라지게 하는 UI입니다.

### 접근 방법

```csharp
WarningUI.Instance.Show("인벤토리가 가득 찼습니다!");
```

### 동작 원리

1. `Show(message)` 호출 시 텍스트를 설정하고 알파값을 1로 만듭니다.
2. `displayDuration`초 동안 표시합니다.
3. `fadeDuration`초에 걸쳐 알파값이 0으로 페이드아웃됩니다.
4. 이전 Show가 진행 중이면 기존 트윈을 취소하고 새로운 것을 시작합니다.

### 사용처

- `PlayerMiner` — 인벤토리가 가득 찼을 때 경고 표시

## MoneyItem.cs (마커 컴포넌트)

돈 아이템에 부착하여 **"이것은 돈 아이템이다"** 라고 식별하는 컴포넌트입니다.

### 필드

| 필드 | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `value` | int | 1 | 이 아이템의 화폐 가치 |

### 사용 방식

```csharp
// PickupZone에서 아이템을 주울 때:
MoneyItem moneyItem = item.GetComponent<MoneyItem>();
if (moneyItem != null) {
    CurrencyManager.Instance.Add(moneyItem.value);
}
```

- `PickupZone`이 아이템을 줍는 과정에서 `GetComponent<MoneyItem>()`으로 확인합니다.
- MoneyItem이 있으면 `CurrencyManager`에 금액을 추가합니다.
- 돈 프리팹에만 이 컴포넌트를 붙이면 됩니다.
