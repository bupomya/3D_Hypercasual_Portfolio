# 가공 & 교환 시스템

광물을 가공하고 NPC와 교환하여 돈으로 바꾸는 시스템입니다.

```
ConveyorLine      ← 아이템을 웨이포인트 따라 이동시키며 변환
ExchangeManager   ← NPC를 소환하여 빨간 광물 → 돈 교환
```

---

## ConveyorLine.cs

DropZone에 쌓인 아이템을 **웨이포인트를 따라 이동시키며 다른 아이템으로 가공**합니다.

### 주요 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `sourceDropZone` | DropZone | 아이템을 꺼낼 드롭 존 |
| `moveTarget` | Transform | 첫 번째 이동 목적지 |
| `waypoints` | Transform[] | 순서대로 거칠 경유 지점들 |
| `moveSpeed` | float | 아이템 이동 속도 (기본값 3) |
| `sendInterval` | float | 아이템 간 처리 간격 (기본값 0.5초) |
| `convertedPrefab` | GameObject | 변환 후 생성할 프리팹 |
| `outputStacker` | GridStacker | 최종 출력 위치 |
| `dropTweenDuration` | float | 마지막 드롭 애니메이션 시간 |
| `dropTweenHeight` | float | 마지막 드롭 포물선 높이 |

### 처리 흐름

```
[1] DropZone의 GridStacker에서 아이템 PopItem()
         │
         ▼
[2] moveTarget 위치로 이동 (moveSpeed 속도)
         │
         ▼
[3] 원본 아이템 파괴 → convertedPrefab 생성 (변환!)
         │
         ▼
[4] waypoints[0] → waypoints[1] → ... 순서대로 이동
         │
         ▼
[5] outputStacker에 ItemTweenHelper.MoveArc()로 도착
         │
         ▼
[6] sendInterval 대기 후 [1]부터 반복
```

### 대기 조건

- `sourceDropZone.HasPendingTweens == true`이면 대기 — 아직 DropZone에 날아오는 아이템이 있습니다.
- 소스에 아이템이 없으면 대기합니다.

### 에디터 시각화

Scene 뷰에서 **노란색 선**으로 moveTarget → waypoints → outputStacker 경로를 보여줍니다.

---

## ExchangeManager.cs

NPC를 소환하여 **빨간 광물을 돈으로 교환**하는 시스템입니다.
전체 과정이 하나의 코루틴으로 무한 반복됩니다.

### 주요 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `npcPrefab` | GameObject | 소환할 NPC 프리팹 |
| `spawnPoint` | Transform | NPC 생성 위치 |
| `waitingPoint` | Transform | NPC가 대기할 위치 |
| `arrivePoint` | Transform | NPC가 퇴장할 위치 |
| `npcMoveSpeed` | float | NPC 이동 속도 (기본값 3) |
| `mineralRedDrop` | GridStacker | 입력: 빨간 광물이 쌓여 있는 곳 |
| `requiredCount` | int | 교환에 필요한 빨간 광물 수 (기본값 3) |
| `feedInterval` | float | 광물을 NPC에게 먹이는 간격 (기본값 0.3초) |
| `moneyDrop` | GridStacker | 출력: 돈 아이템이 생성될 곳 |
| `moneyPrefab` | GameObject | 생성할 돈 아이템 프리팹 |
| `moneyPerExchange` | int | 1회 교환 시 받는 돈 수 (기본값 3) |
| `moneySpawnInterval` | float | 돈 아이템 생성 간격 (기본값 0.2초) |

### 교환 라이프사이클 (코루틴)

```
[1] NPC를 spawnPoint에 Instantiate
         │
         ▼
[2] NPC가 waitingPoint로 이동 (Vector3.MoveTowards)
         │  (빨간 광물이 requiredCount개 쌓일 때까지 대기)
         ▼
[3] 빨간 광물을 하나씩 NPC에게 먹임
         │  (아이템이 포물선으로 NPC에게 날아가 파괴됨)
         │  (feedInterval 간격으로 반복)
         ▼
[4] 돈 아이템을 moneyDrop에 생성
         │  (moneyPerExchange개, moneySpawnInterval 간격)
         ▼
[5] NPC가 arrivePoint로 이동
         │
         ▼
[6] NPC 파괴 (Destroy)
         │
         └→ [1]로 돌아가 무한 반복
```

### 핵심 포인트

- NPC 이동은 `Vector3.MoveTowards()`를 사용합니다.
- 아이템 → NPC 애니메이션은 `ItemTweenHelper.MoveArc()`를 사용합니다.
- 빨간 광물이 부족하면 [3] 단계에서 채워질 때까지 대기합니다.
