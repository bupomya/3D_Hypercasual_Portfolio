# Hypercasual Mining Game - 코드 가이드

> 이 문서는 프로젝트에 처음 참여하는 개발자를 위한 코드 흐름 및 컴포넌트 설명서입니다.

---

## 게임 개요

광물을 채굴하고, 가공·교환하여 돈을 벌고, 업그레이드를 통해 성장하는 **하이퍼캐주얼 마이닝 게임**입니다.

**핵심 루프:** 채굴 → 아이템 드롭 → 컨베이어 가공 → NPC 교환 → 돈 획득 → 레벨업 → 더 강력한 채굴

---

## 폴더 구조

```
Assets/
├── Scripts/            ← 모든 C# 게임 스크립트 (핵심 코드)
├── Prefabs/            ← 프리팹 (Mineral, Worker, MoneyItem 등)
├── ScriptableObject/   ← 레벨별 설정 데이터 (MiningLevelData)
├── Scenes/             ← 게임 씬 (SampleScene)
├── Animations/         ← 캐릭터 애니메이션
├── Plugins/DOTween/    ← 트윈 애니메이션 라이브러리
├── Plugins/VirtualJoystick/ ← 모바일 조이스틱 입력
└── 0.Assets/           ← 외부 에셋팩
```

---

## 게임 흐름 (전체 사이클)

```
[게임 시작]
    │  가이드 화살표 → MiningZone 방향
    ▼
플레이어가 MiningZone에 진입
    │
    ├─ 레벨 0: 플레이어가 직접 채굴
    └─ 레벨 1+: Worker(NPC)가 대신 채굴
    │
    ▼
채굴된 광물이 플레이어의 ItemStacker에 쌓임
    │ (가득 차면 경고 UI 표시)
    │  가이드 화살표 → MineralDropZone 방향
    ▼
DropZone에 진입 → 광물이 GridStacker로 이동 (maxCount 제한 가능)
    │  가이드 화살표 → MineralRedPickupZone 방향
    ▼
ConveyorLine이 광물을 가공 (일반 광물 → 빨간 광물)
    │  빨간 광물 획득 시 가이드 화살표 → ExchangeZone 방향
    ▼
ExchangeManager가 NPC를 불러 빨간 광물 → 돈으로 교환
    │  가이드 화살표 → MoneyItem 위치
    ▼
PickupZone에서 돈 아이템을 주워 CurrencyManager에 반영
    │  MoneyItem 획득 → 가이드 완료, LevelUpZone 카메라 연출 + 활성화
    ▼
LevelUpZone에 진입 → 돈을 소비하여 레벨업
    │
    └─ 모든 채굴 관련 수치가 업그레이드됨
       (채굴 데미지, 보유량, 자동 채굴 간격 등)

[ArriveZone 가득 참]
    │
    ▼
ArriveZoneExpansionZone 카메라 연출 + 활성화
    → 비용 지불 시 수용인원 확장 + 벽 애니메이션
```
